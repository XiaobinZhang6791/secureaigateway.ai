## Core idea

AI governance should not stay only in policy documents.

A practical starting point is one governed AI access path:

- Who is using AI?
- For what business purpose?
- What data is involved?
- Which model is used?
- What policy decision is applied?
- What audit evidence is recorded?
- Can usage and token cost be explained later?
