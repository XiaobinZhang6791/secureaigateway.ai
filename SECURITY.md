# Security Policy

This repository contains public information about SecureAI Gateway.

It does not contain production source code, customer data, deployment secrets, API keys or internal security configurations.

## Reporting security issues

If you believe you have found a security issue related to SecureAI Gateway public resources or the official website, please contact:

contact@secureaigateway.ai

Please do not include sensitive customer data, secrets, credentials or private information in public GitHub issues.
